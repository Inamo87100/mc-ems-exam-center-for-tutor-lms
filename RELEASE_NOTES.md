# MC-EMS Base v1.0.0 Release Notes

## Features

- **Exam Sessions CPT**: Create and manage exam sessions effectively.
- **Candidate Booking Calendar**: A calendar feature for candidates to track their bookings.
- **Manage Bookings Shortcode**: A shortcode to manage bookings directly on pages.
- **Advanced Bookings List**: An extensive list of bookings with search functionality and CSV export capacity.
- **Tutor LMS Integration**: Integrated with Tutor LMS for seamless management.
- **Proctor Assignment Calendar**: Assign proctors easily with a dedicated calendar.
- **CSS and JavaScript Assets**: Enhanced styling and functionality through optimized assets.
- **i18n Support**: Full internationalization capabilities for global usage.
